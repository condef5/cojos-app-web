## Setup development

1. You need these programs:

- [NodeJs](https://nodejs.org/en/)
- [yarn](https://yarnpkg.com/)

2. Run migrations and seeds

```sh
yarn setup
```

3. Start dev server:

```sh
yarn dev
```

4. Go to http://localhost:3000/
